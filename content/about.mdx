# About

Yorick Docs is a community driven documentation project that provides clear, step-by-step guides for setting up and managing FiveM servers.

## What's Covered

- **FXServer**: Windows and Linux server creation, configuration, and maintenance
- **txAdmin**: Account management, bot setup, and server administration
- **Database**: MySQL/MariaDB setup via XAMPP and standalone installations
- **Security**: Backdoor detection, sandboxing, and server hardening
- **General**: Development tools and environment setup
