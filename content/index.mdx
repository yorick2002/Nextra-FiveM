## Platform Support

These guides cover both **Windows** and **Linux** server configurations. Each platform section includes:

- **Server creation**: From download to first launch
- **Database setup**: MySQL/MariaDB configuration via XAMPP or standalone
- **Resource management**: Installing and configuring server resources
- **Port forwarding** (Windows) / **Domain setup** (Linux): Making your server accessible
- **Auto-updates** (Linux): Keep your server current automatically
